## Building

Requires: `pdflatex` with `tcolorbox`, `tikz`, `enumitem`, `booktabs`, `tabularx`, `fancyhdr`, `hyperref` packages.

```bash
cd /path/to/repo
pdflatex megn301_master_reference.tex
pdflatex megn301_master_reference.tex  # run twice for TOC
```
